# 포크덩 (Pokdeng) 소셜 멀티플레이어 웹앱

태국 카드게임 포크덩을 실제 현금 없이, 🪙칩(베팅용)과 💎다이아(스토어 결제 모의)만으로
즐기는 실시간 멀티플레이어 웹앱입니다.

## 실행 방법

```bash
npm install
npm start
```

브라우저에서 http://localhost:3000 접속. 여러 탭/창을 열면 여러 플레이어로 테스트할 수 있습니다.

## 주요 구성

- `server.js` — Express + Socket.io 서버. 로비/방 관리, 베팅·턴 진행, 마켓(다이아 구매·교환·무료 칩),
  재접속 처리(브라우저에 저장된 고유 clientId로 새로고침해도 지갑/방 복귀)를 담당합니다.
- `game/pokdeng.js` — 포크덩 카드 판정 순수 함수 모음 (팍 8/9, 통, 시안, 리앙, 배당 계산 등).
- `public/index.html` / `style.css` / `client.js` — 4탭(홈/포크덩/마켓/마이페이지) 프론트엔드,
  오벌 테이블 UI, WIN 리본 + 결과 모달, Web Audio 사운드 이펙트, 잔액 카운트업 애니메이션.

## 게임 규칙 (구현된 버전)

- 각 플레이어는 2장을 받고, 합이 8 또는 9면 "팍"으로 즉시 승부가 결정됩니다 (드로우 없음).
- 팍이 없으면 순서대로 드로우/스테이를 선택해 3번째 카드를 받을 수 있습니다.
- 뱅커는 모든 플레이어의 선택이 끝난 뒤 마지막으로 드로우/스테이를 선택합니다.
- 배당: 통(같은 숫자 3장) 5배, 시안(같은 무늬 K·Q·J) / 리앙(같은 무늬 3연속) 3배,
  팍9 2배, 팍8/일반 1배.
- 매 라운드가 끝나면 뱅커가 다음 활성 플레이어로 자동 로테이션됩니다.

## 로비 / 베팅 단계

연습(무료) → 브론즈(100) → 실버(500) → 골드(2,000) → 플래티넘(10,000) → VIP(50,000) 🪙,
각 단계별로 여러 개의 방이 있으며 빠른 매칭 또는 방을 직접 선택해 입장할 수 있습니다.

## 참고

- 다이아 구매는 실제 결제가 이뤄지지 않는 모의(mock) 처리입니다. 실제 앱스토어/구글플레이 IAP
  연동은 별도 작업이 필요합니다.
- 데이터는 서버 메모리에만 저장됩니다(서버 재시작 시 초기화). 영구 저장이 필요하면 DB 연동을
  추가하세요.
